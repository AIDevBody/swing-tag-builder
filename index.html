Tag Builder for Source Calendars

Language:
How to use
  1. Choose your preferred language from the top-right selector. English is the fallback; if a translation is missing, the raw tag is shown.
  2. Browse each tag category and select the options using the shown UI (dropdowns or chips).
  3. Your selections are mirrored in the textbox below. Click Copy tags to copy them.
  4. Click Create the event to open Google Calendar with your description plus the tags appended, separated by a line of ----------.
